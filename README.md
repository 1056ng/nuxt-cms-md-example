# Nuxt CMS EXAMPLE
example for [nuxt-cms-md](https://github.com/1056ng/nuxt-cms-md)

## Structure

```
nuxt-cms-md-example
├── nuxt.config.js
├── package.json
├── models
│   └── blogs
│       ├── apple.md
│       └── grape.md
│       └── kiwi.md
│       └── lemon.md
│       └── images
│           └── ....
└── pages
    └── index.vue
    └── blogs
        └── _id.md
```

## HOW TO USE

```
npm i
npm start
```
